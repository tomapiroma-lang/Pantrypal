უსაფრთხოება

ბოლო განახლება: 2026 წელი

1. პაროლის დაცვა

თქვენი პაროლი ინახება bcrypt ალგორითმით დაშიფრული სახით. ჩვენ არასდროს ვხედავთ თქვენს პაროლს plain text-ად.

  • გამოიყენეთ მინიმუმ 8 სიმბოლო
  • შეიტანეთ დიდი და პატარა ასოები, ციფრები
  • ნუ გამოიყენებთ ერთ პაროლს სხვადასხვა საიტზე

2. CSRF დაცვა

ყველა ფორმა დაცულია CSRF (Cross-Site Request Forgery) შეტევებისგან Flask-WTF-ის hidden token-ის საშუალებით.

3. სესიების მართვა

სისტემაში შესვლის სესიები მართულია Flask-Login-ით. "დამიმახსოვრე" ფუნქცია 30 დღის განმავლობაში ინარჩუნებს სესიას. გასვლისას სესია სრულად იწმინდება.

4. API Key-ების დაცვა

Groq API key და სხვა სენსიტიური გასაღებები ინახება სერვერზე environment variable-ებში და არასდროს არ ჩანს კლიენტის მხარეს.

5. SQL Injection დაცვა

მონაცემთა ბაზასთან ყველა ურთიერთობა ხდება SQLAlchemy ORM-ის საშუალებით, რაც SQL Injection შეტევებს გამორიცხავს.

6. Rate Limiting

Groq API-ზე მოთხოვნები იზღუდება caching-ის საშუალებით, რათა თავიდან ავიცილოთ ბოროტად გამოყენება და ზედმეტი ხარჯები.

7. პრობლემის შეტყობინება

თუ უსაფრთხოების პრობლემა აღმოაჩინეთ, გთხოვთ დაუყოვნებლივ დაგვიკავშირდეთ: security@pantrypal.ge
