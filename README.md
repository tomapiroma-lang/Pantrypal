PantryPal 🍳
PantryPal არის ჭკვიანი კულინარიული კომპასი, რომელიც მომხმარებლებს ეხმარება იპოვონ იდეალური რეცეპტები იმ ინგრედიენტების მიხედვით, რომლებიც მათ უკვე აქვთ მაცივარში. ტრადიციული კულინარიული პლატფორმებისგან განსხვავებით, ეს აპლიკაცია ორიენტირებულია პრაქტიკულობაზე, დროის დაზოგვასა და საკვების ნარჩენების შემცირებაზე.

🚀 პროექტის არსი და მთავარი იდეა
"რა მოვამზადო იმით, რაც მაცივარში მაქვს?" — ეს არის კითხვა, რომელსაც PantryPal პასუხობს. მომხმარებელი მარტივად მონიშნავს მის ხელთ არსებულ პროდუქტებს (მაგალითად: კვერცხი, ყველი, ტომატი), ალგორითმი კი მყისიერად სთავაზობს იმ რეცეპტების სიას, რომელთა მომზადებაც იმავე წამს, დამატებითი საყიდლების გარეშეა შესაძლებელი.

✨ რით არის განსხვავებული და უნიკალური?
საკვების ნარჩენების (Food Waste) შემცირება: აპლიკაცია ებრძვის გლობალურ პრობლემას და მომხმარებელს ეხმარება მაქსიმალურად გამოიყენოს სახლში არსებული პროდუქტები, სანამ მათ ვადა გაუვათ.

დროისა და ფულის დაზოგვა: აღარ არის საჭირო რეცეპტების ხანგრძლივი ძებნა ან მარკეტში მოულოდნელი ვიზიტები ერთი კონკრეტული ინგრედიენტის გამო.

💰 მონეტიზაციის მოდელი
PantryPal-ის ბიზნეს მოდელი ეფუძნება წამყვან სუპერმარკეტების ქსელებთან პარტნიორობას. თუ მომხმარებელს შერჩეული რეცეპტისთვის აკლდება რომელიმე კომპონენტი, პლატფორმა მას სთავაზობს "ერთი კლიკით გამოძახების" ფუნქციას, რითაც პარტნიორი მარკეტიდან საჭირო პროდუქტების ლოგისტიკა და მიწოდება მომენტალურად ხდება.

🛠️ ტექნოლოგიური სტეკი
Backend: Python (Flask ფრეიმვორკი) ბიზნეს ლოგიკისა და რეცეპტების ფილტრაციისთვის.

Frontend: HTML5, CSS3 (სუფთა ანიმაციებითა და სტრუქტურით) და Bootstrap საიტის სრული ადაპტურობისთვის.

Deployment: პროექტი ოპტიმიზებულია მარტივი ჰოსტინგისთვის და გაშვებულია შესაბამის პლატფორმაზე.
